# Fungus In Forest

Fungus In Forest is a Fabric mod for Minecraft 1.21.1 that adds small mushroom clusters to Overworld forest, taiga, and jungle biomes.

## Features

- Adds visible brown mushroom clusters to Overworld forest, taiga, and jungle biomes.
- Adds smaller red mushroom clusters to Overworld forest, taiga, and jungle biomes.
- Clusters are small, usually around 4-5 mushrooms when the terrain can support them.
- World generation can place these mushrooms in full light.
- Keeps the change lightweight and vanilla-friendly.

## Build

Requirements:

- Java 21.
- Gradle, or an IDE with Gradle support.

Command:

```powershell
gradle build
```

The playable jar is available in `build/libs/`.
