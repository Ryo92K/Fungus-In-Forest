# Fungus In Forest

Fungus In Forest is a Fabric mod for Minecraft 1.21.1 that adds small mushroom to forest, taiga, and jungle biomes.


## Features

- Adds visible brown mushroom to Overworld forest, taiga, and jungle biomes.
- Keeps the change lightweight and vanilla-friendly.
